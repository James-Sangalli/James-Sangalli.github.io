JavaScript Basics

How does JavaScript compare to HTML and CSS?

HTML is a markup language and CSS styles up the markup. Technically HTML and CSS are not programming languages because they are only templates. JavaScript is an actual programming language that typically runs client side (on your browser) and unlike HTML and CSS, JavaSript can perform functions and methods but can also be used to add details to the visual side of your webpage with things like jQuery.

What is control flow and loops?

In programming we use control flows to perform decisions based on what data we get. an example of this is the if/else statement which executes a section of code based on what is inputed. A real life example of this could be if you were deciding on whether to walk or drive to work. You could say for example, if it is not raining tommorow then walk else if it is raining then drive. loops are also used to iterate through data. An example of this is a while loop which states that it will go through something until a condition is reached. A real life example of this could be while the time is less than 6 o'clock I will continue working. Once it is 6 o'clock however you can then take a break. If a loop has a condition that is never met it will generate an infinite loop and crash the program.

What is difference between accessing data from arrays and object literals?

When accessing data from Object Literals you must use either dot notation or bracket notation to target the exact section of data you want e.g. item.subitem. When you access data from an array you must place the specific element in square brackets [] so that the computer knows which element of data you wish to use from the array.

What are functions and why they are useful?

Functions are basically sections of code that execute a task. For example you could have a function that when called takes in two numbers, multiplies them together and returns the result.
