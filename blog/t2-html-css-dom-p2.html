What is grid based design?

A: Grid Based design is a method of dividing the page up into collumns and designing each column in its fraction. This allows for an easy way to divide up the page and design the desired sections. There are typically 12 columns per page

Whats all the hype about responsive webpages?

A: In todays world there are many devices ranging from Destop's, laptops , smart phone and tablets. Each device has a different screen size and thus there is a need to have your webpage adapt to each device type so that your users can navigate your pages easily. While it is still possile to view a desktop page properly on a small device it is more difficult to use

Semantic structure

A:Semantic structure is laying out your webpage appropriately to convey the user to see important information and easily navigate the site. For example if you have an important article on a news site you would want a grabbing title followed by the text under it so that your users can read it easily.
