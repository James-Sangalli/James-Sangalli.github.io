Explain .map(), .filter() and .reduce()

.map() performs a similar function to a for loop. It can take a value and alter it based on what it is programmed to do expect without the need for a for loop. If for example you wanted to do something simple like add 1 to each number parsed from an array you could use .map instead to add it without having to use the for loop. .filter() typically returns a boolean value on an array rather than returning the array itself, it is like a checker to see if the array element equals what was specified in the parenthesis. .reduce() takes two parameters and uses them to iterate through an array returning only one (the final result). It basically iterates through an array based on a starting point (first parameter) and assigns the result to the other variable. An example of this could be totaling up the scores of the pupils in a class room whereby the starting point would be the first student and the other variable being the total result of all the students.
