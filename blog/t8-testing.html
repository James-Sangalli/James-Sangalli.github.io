What is scope in javascript?

In JavaScript, scope is either global or local. What this means is that if you declare a variable within a function it is local in scope it can only be used within the function. A globally declared variable however is declared outside of any function and can be referenced or changed anywhere within the code and is thus considered a global scope variable.
