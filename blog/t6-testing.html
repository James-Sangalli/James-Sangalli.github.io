Describe parentheses ()

Parentheses are typically used on functions to parse in parameters. If parentheses are empty then nothing is to be parsed but when you call the function you still include them. Some functions do require parameters and if so you must call the function with the appropriate variable(s) being placed within the parentheses.

Describe Brackets []

In javascript we use brackets when trying to index arrays (get a specific element(s) from an array) or to contain multiple index's within an object.

Describe Braces {}

Braces are used in javascript to contain functions, if statements and many other code blocks. It is useful because it tells the computer and the programmer where one specific code block begins and when it ends. The if statement is an example of this, you put if(something){Something} else {something}. The braces tell the compiler where the if statement code block is to be executed and when it ends.

Describe single quotes '' and double quotes ""

In JavaScript single and double quotes both do the same thing. However for JSON and for variables that have apostrophe's it is best to use double quotes.
