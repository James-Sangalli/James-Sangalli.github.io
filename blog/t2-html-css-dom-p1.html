Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

A: HTML is like the basic structure of a website with all the containers to hold it together, CSS is like the decorator for a webpage and makes those containers look good and DOM is simply the structure of the Page and how it interacts with each other in a tree structure e.g. within the head of the document there is a header h1

What is meant by boxifying design?

A: Boxifying design is the method of placing each significant item displayed on your page into boxes. By doing this you have more control over each item and the ability to easily edit your page.

What is the box model?

A:each element on an HTML page is divided into a box regardless of its shape. This is the box model and each box contains a border, padding, margin and content. The content part of the box contains all the information like text and pictures, the padding is a transparent layer outside the content, the border is simply a border going round the content and padding and lastly the margin clears space outside of the border
